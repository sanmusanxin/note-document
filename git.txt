git 使用

将本地新增的文件同步到远程仓库：
1. 确保本地仓库的根目录下打开终端或命令行界面。

2. 使用以下命令将新增的文件添加到 Git 的暂存区：
   git add <文件名>
   如果想将所有新增的文件都添加到暂存区，可以使用：
   git add .

3. 使用以下命令提交更改到本地仓库：
   git commit -m "添加新文件"
   替换 `"添加新文件"` 为自己的提交信息。

4. 最后，使用以下命令将本地的提交推送到远程仓库：
   git push origin <分支名>
   替换 `<分支名>` 为要推送的分支的名称，通常是 `main` 或 `master`。
